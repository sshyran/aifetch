# AI Fetch

Or *I* fetch

## Installation

*(Unix only)*

With Node.js installed,

```bash
npm install -g aifetch
```

Make sure [wget](https://www.gnu.org/software/wget/) is installed as well.

```bash
# macOS
brew install wget

# Ubuntu
apt-get update && apt-get install wget

# etc. 
```

## Usage

Run the command in your shell

```
aifetch
```
